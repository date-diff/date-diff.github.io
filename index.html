<date-diff> Web Component

A simple Web Component that displays the difference between two dates in days or years.

Basic Examples

Days from a past date to today:
<date-diff from="2024-01-01"></date-diff>
Days from today to a future date:
<date-diff to="2024-12-31"></date-diff>
Days between two specific dates:
<date-diff from="2024-01-01" to="2024-12-31"></date-diff>

Using Years

Years from birth date to today:
<date-diff from="1990-01-01" years></date-diff>
Years between two dates:
<date-diff from="2000-01-01" to="2024-01-01" years></date-diff>

Edge Cases

No attributes (0 days - today to today):
<date-diff></date-diff>
Negative difference (past date):
<date-diff from="2025-01-01" to="2024-01-01"></date-diff>
Explicit days attribute:
<date-diff from="2024-01-01" to="2024-01-31" days></date-diff>

Usage

Include the script in your HTML:

<script src="date-diff.js"></script>

Then use the component:

<date-diff from="2024-01-01" to="2024-12-31"></date-diff>

Attributes

  • from: Start date (ISO format: YYYY-MM-DD). Defaults to today.
  • to: End date (ISO format: YYYY-MM-DD). Defaults to today.
  • years: Display difference in years (boolean attribute)
  • days: Display difference in days (boolean attribute, default)
